# Hướng dẫn điều hướng trang web (Website Navigation)
Khi khách hàng hỏi cách truy cập các trang hoặc chức năng trên website, hãy hướng dẫn họ sử dụng các đường dẫn (URL) sau:

- **Trang chủ**: `/`
- **Danh sách sản phẩm**: `/product`
- **Giỏ hàng**: `/cart`
- **Thanh toán**: `/checkout`
- **Đăng nhập**: `/login`
- **Đăng ký**: `/register`
- **Quên mật khẩu**: `/forgot-password`
- **Lịch sử đơn hàng**: `/order-history`
- **Thông tin tài khoản (Profile)**: `/user-profile`
- **Sản phẩm yêu thích**: `/favorites`
- **Theo dõi đơn hàng**: `/track`

*Lưu ý*: Luôn chèn link bằng định dạng Markdown, ví dụ: "Anh/chị có thể [Đăng nhập](/login) để tiếp tục."
